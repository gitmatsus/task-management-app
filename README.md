# タスク管理アプリ (Modern To-Do App)

ブラウザひとつで完結する、軽量かつ高機能なシングルページ・タスク管理アプリケーションです。HTMLファイルを開くだけですぐに使用を開始できます。

## 🚀 主な機能

* **マルチリスト管理**: 「仕事」「プライベート」など、用途に合わせた複数のタスクリストを作成・切り替え可能です。
* **直感的な操作 (Drag & Drop)**:
    * タスクの並び替え（優先順位の調整）
    * タスクをゴミ箱ボタンへドラッグして削除
    * ゴミ箱内のタスクを「未完了/完了」フィルターへドラッグして直接復元
* **期限管理**: タスクに期限を設定でき、「今日」「明日」「期限切れ」を視覚的に判別可能なバッジで表示します。
* **ダークモード対応**: システム設定や好みに合わせて、洗練されたダークテーマに切り替え可能です。
* **動的なステータス表示**: リストごとのタスク数や、期限の状態（期限切れ、本日分など）をカラードットでリアルタイムに表示します。
* **データ永続化**: ブラウザの `localStorage` を使用しているため、ページを閉じてもデータが保持されます。

## 🛠 技術スタック

* **Frontend**: Vanilla JavaScript (ES6+), HTML5, CSS3
* **Icons**: SVGベースのカスタムアイコン
* **Storage**: Web Storage API (localStorage)
* **Dependency**: 外部ライブラリ（jQuery, React等）への依存なし

## 📖 使い方

### 1. タスクの追加・編集
* 上部の入力フォームに内容を入力し、「追加」ボタンまたは **Enterキー** で登録します。
* タスク名を **ダブルクリック** することで、内容の書き換えや期限の設定・変更が可能です。

### 2. リストの管理
* 左側のサイドバーでリストを切り替えます。
* リスト名を **ダブルクリック** すると名前を変更できます。
* 「＋リストを追加」から新しいカテゴリを作成できます。

### 3. ドラッグ＆ドロップ
* タスク左側の「グリップアイコン」を掴んで自由に並び替えができます。
* 不要なタスクはゴミ箱へドラッグするか、タスク右側のゴミ箱ボタンをクリックして削除します。

### 4. 復元と完全削除
* 画面右下の「ゴミ箱」ボタンから削除済みアイテムを確認できます。
* 「復元」ボタンで元のリストに戻すか、アイテムをドラッグして特定のステータス（完了/未完了）として復元できます。

## 📝 開発者向け
このアプリは `app.html` という1つのファイルに全てのロジック、スタイル、マークアップが含まれています。サーバーを立てる必要はなく、ローカル環境でのタスク管理ツールとして、あるいはWebアプリのベースとして活用いただけます。
